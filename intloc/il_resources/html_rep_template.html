$int_table_head$

Integration table

$int_table_data$ @insert@@ $int_table_foot$
ID Chr/contig loc +/-[bp] supp. int. reads non-int. reads ts_cov IMR SSBR
#

loc = location/position on chromosome in basepairs

+/-[bp] = location placement uncertainty in basepairs (rounded SD of alignment edges used to determine insertion location)

supp. int. reads = reads containing int.-seq. supporting integration at site

non-int. reads = reads covering genomic location of integration site not containing int.-seq. (e.g. originating from homolog. chrom.)

ts_cov = total number of reads covering integration site

IMR = identity match ratio (the average identity of all non-construct sequences in all supporting reads to the integration site)

SSBR = most similar secondary site bitscore ratio (ratio of summed bitscores of supporting reads aligned to the called int. site or to the next best potential alternative site)
